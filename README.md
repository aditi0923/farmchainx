🧑‍🌾 FarmchainX – AI Powered Transparent Agricultural Marketplace

FarmchainX is a full-stack web application that connects farmers directly with consumers, eliminating middlemen and ensuring fair pricing, transparency, and trust using AI and Blockchain technologies.

🚀 Features

🌾 Direct farmer-to-consumer marketplace
🤖 AI-based price prediction and demand forecasting
🔗 Blockchain-based product traceability
📦 Supply chain and logistics tracking
💳 Secure payment integration (Razorpay)
🌐 Multi-language support
📱 QR code verification for product authenticity
❤️ Favorites and direct checkout system

🛠️ Tech Stack

1.Frontend
a.React.js
b.HTML, CSS, JavaScript
2.Backend
a.Java Spring Boot
3.Database
a.MySQL
4.Tools & Integrations
a.Razorpay (Payments)
b.Postman (API Testing)

🏗️ System Architecture
1.User Interface Layer (React)

4.AI Engine (Prediction & Analytics)
5.Blockchain Layer (Traceability)

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/aditi0923/farmchainx.git
cd farmchainx
2️⃣ Frontend Setup
cd FarmchainX
npm install
npm run dev
3️⃣ Backend Setup
cd backend
./mvnw spring-boot:run
4️⃣ Environment Variables

Create .env file in backend:

DB_URL=your_database_url
DB_USERNAME=root
DB_PASSWORD=your_password

RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret

🔄 Workflow

1.Farmer registers and logs in
2.Adds product details
3.AI suggests price
4.Customer browses products
5.Places order and makes payment
6.Blockchain records transaction
7.QR code verifies product
8.Delivery is tracked
